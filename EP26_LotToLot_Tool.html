1 Protocol Setup — CLSI EP26 Stage 1
Purpose: This tab establishes the protocol parameters for each analyte (one-time setup). Enter analyte data, then the tool calculates the Rejection Limit (RL) and required sample size (N) by looking up the embedded CLSI EP26 Appendix A tables (A1, A2, A3) based on CD/SWRL, Sr/SWRL, and desired statistical power. The output checklist is used for all future lot evaluations.
2 Step 1: Download Input Template

Download and complete the CSV template with your analyte parameters. Fill one row per analyte concentration level, then import it below.

3 Step 2: Enter or Import Analyte Data
Instructions:
  • CD: Critical Difference (absolute units or %TEa-based, same units as SWRL)
  • SWRL: Within-Reagent-Lot imprecision (total, SD)
  • Sr: Repeatability / within-run SD
  • Levels: Number of medical decision concentrations being tested (1, 2, or 3 → selects Table A1, A2, A3)
  • Power Goal: Desired statistical power (0.80, 0.85, 0.90, or 0.95)

4 Step 3: Calculate Protocol Parameters
Calculations use the embedded CLSI EP26 Appendix A tables. The tool computes CD/SWRL and Sr/SWRL ratios, selects the appropriate table (A1 for 1 level, A2 for 2, A3 for 3), and returns: Rejection Limit (RL) = k × CD, Number of Samples (N), and achieved Power.
