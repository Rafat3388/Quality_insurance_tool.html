❓ How to Use

📋 Step-by-Step Guide:

  1. Enter Analyte Information: Name, unit, and distribution type
  2. Input Biological Variation Data: Get values from EFLM database or use the BV Database tab
  3. Add Reference Values: Click "+ Add Value" or import from CSV file
  4. Calculate: Click "Calculate Reference Interval" button
  5. View Results: Switch to Results tab to see calculations
  6. Generate Report: Go to Report tab to download or print

💡 Tips:

  • Minimum 6 reference values required (16+ recommended)
  • Use validated BV data from EFLM database
  • This tool works completely offline - save the HTML file
  • Data is saved in your browser automatically
  • Works on PC, tablet, and mobile devices

📚 Method Reference:

Coşkun A, Sandberg S, Unsal I, Topcu DI, Aarsand AK. Reference Intervals Revisited: A Novel Model for Population-Based Reference Intervals, Using a Small Sample Size and Biological Variation Data. Clinical Chemistry 2024;70(10):1279-1290.

🔄 Offline Use:

Save this HTML file to your computer or mobile device. It will work without internet connection. All calculations are performed locally in your browser.
